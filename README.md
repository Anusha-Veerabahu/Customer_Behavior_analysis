# Customer Shopping Behavior Analytics Project

## Overview

This project focuses on analyzing customer shopping behavior using a complete data analytics workflow. The process includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard creation, report generation, and presentation development. The goal is to uncover customer purchasing patterns, product performance, and business insights that support data-driven decision-making.

---

## Dataset

The dataset contains customer shopping information, including:

* Customer ID
* Age
* Gender
* Category
* Item Purchased
* Purchase Amount (USD)
* Review Rating
* Subscription Status
* Payment Method
* Shipping Type
* Discount Applied
* Promo Code Usage
* Previous Purchases

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Verified data structure and data types.
* Performed initial exploration of records and columns.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data formats.
* Prepared data for analysis.

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics.
* Identified purchasing trends.
* Evaluated category-wise performance.
* Studied customer ratings and subscription behavior.

### 4. SQL Analysis

Performed analytical queries to:

* Calculate total revenue.
* Identify top-selling products.
* Analyze category-wise sales.
* Compare customer segments.
* Generate business insights.

### 5. Power BI Dashboard

Created an interactive dashboard featuring:

* Total Customers KPI
* Average Purchase Amount KPI
* Average Review Rating KPI
* Revenue by Category
* Sales by Category
* Revenue by Item Purchased
* Sales by Item Purchased
* Dynamic Filters and Slicers

### 6. Report Creation

* Summarized analytical findings.
* Documented key business insights.
* Prepared professional project documentation.

### 7. Presentation Development

* Created a project presentation using Gamma AI.
* Included objectives, methodology, dashboard screenshots, and insights.

## Key Insights

* Clothing generated the highest revenue among product categories.
* Subscription customers contributed significantly to overall sales.
* Customer ratings remained consistently positive across categories.
* Product categories showed varying purchasing trends that can guide inventory planning.

## How to Run

### Python Analysis

1. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

2. Run the analysis script:

   ```bash
   python analysis.py
   ```

### SQL Analysis

1. Import the dataset into PostgreSQL/MySQL/SQL Server.
2. Execute the SQL queries provided in the project.

### Power BI Dashboard

1. Open the `.pbix` file.
2. Refresh the dataset.
3. Interact with filters and visualizations.
